## Main
in Weekly Vessel Valuations Report 03/08/2021
Image reference: 2021-08-03_weekly-vessel-valuations-report-august-03-2021_img1.png
Tanker: Modern tonnage has firmed slightly.
VLCC Tsuruga (310,000 DWT, Oct 2009, Imabari) sold to Cyprus Maritime Co for USD 38.00 mil, VV value USD 41.91 mil.
MR2 Grand Ace 2 (46,000 DWT, Mar 2006, STX Offshore) sold to Aerio Shipmanagement for USD 9.30 mil, VV value USD 10.16 mil - SS/DD Passed.
Bulker: Values have remained stable.
Post Panamax Harvest Sky (95,700 DWT, Nov 2013, Imabari) sold to Greek Buyers for USD 23.60 mil, VV value USD 25.35 mil - DD Passed, BWTS fitted.
Post Panamax Dyna Camellia (91,600 DWT, Mar 2007, Imabari) sold to Chinese Buyers for USD 17.50 mil, VV value USD 16.58 mil - BWTS fitted.
Panamax Bulker AM Express (82,200 DWT, Jan 2010, Tadotsu Tsuneishi) sold to Greek Buyers for USD 21.00 mil, VV value USD 20.16 mil.
Ultramaxes Calico Jack (61,000 DWT, Oct 2015, I-S) and Phoenix Rising (60,400 DWT, Jul 2015, Mitsui Ichihara) sold to Greek Buyers in an en bloc for USD 55.00 mil, VV en bloc value USD 55.66 mil - Inc Charter, BWTS fitted.
Supramax Atlantic Merida (56,700 DWT, Jan 2012, Taizhou Kouan) sold to Greek Buyers for USD 15.00 mil, VV value USD 14.28 mil - DD Passed, BWTS fitted.
Supramax Dory (55,700 DWT, Sept 2010, IHI) sold for USD 17.50 mil, VV value USD 17.93 mil - SS/DD Passed, BWTS fitted.
Container: Container values have firmed.
Handy Container Jennifer Schepers (1,794 TEU, Nov 2009, Taizhou Kouan Shipbuilding) sold to MSC for USD 21.00 mil, VV value USD 20.02 mil.
Handy Container Anassa (1,732 TEU, Jul 2006, Guangzhou Wenchong) sold to CMA CGM for USD 20.00 mil, VV value USD 18.60 mil - SS/DD Due.
