## Main
in Weekly Vessel Valuations Report 29/11/2022
Image reference: 2022-11-29_weekly-vessel-valuations-report-november-29-2022_img1.png
Bulkers: Bulker values have remained stable.
Capesize Aquataine (181,700 DWT, Sep 2010, Imabari) sold to unknown Greek buyers for USD 24.75 mil, VV Value USD 25.15 mil - BWTS.
Panamax DL Carnation (81,800 DWT, Sep 2014, Jiangsu Eastern) sold to undisclosed buyers for USD 18.80 mil, VV Value USD 18.68 mil - DD Passed.
Ultramax Italian Bulker (63,500 DWT, Mar 2017, Shin Kasado Dock) sold to DAO Shipping for USD 27.00 mil, VV Value USD 27.52 mil - BWTS.
Tankers: Tanker values have firmed.
Suezmax Concord (159,200 DWT, Jun 2005, Hyundai Heavy Ind) sold to undisclosed buyers for USD 33.00 mil, VV Value USD 28.43 mil - BWTS.
Aframax Nordbay (116,000 DWT, Mar 2007, Universal) sold to undisclosed buyers for USD 34.00 mil, VV Value USD 33.59 mil - BWTS.
MR2 Seabreeze (53,700 DWT, Jul 2007, Shin Kurushima Onishi) sold to undisclosed buyers for USD 19.50 mil, VV Value USD 19.67 mil - SS/DD Passed.
MR2 Atlantica Bridge (50,900 DWT, Dec 2005, STX Offshore) sold to undisclosed buyers for USD 19.75 mil, VV Value USD 19.27 mil - BWTS.
Containers: Container values have softened.
No reported sales.
