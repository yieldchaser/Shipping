## Main
Bulkers : Bulker values have remained stable
Panamax Valiant Spring (81,900 DWT, Dec 2015, Tsuneishi Zhoushan) sold to undisclosed buyers for USD 28.30 mil, VV Value USD 26.94 mil - BWTS.

Tankers : LR values have firmed, whilst VLCC values have softened in the past week
VLCC Princess Mary (306,200 DWT, Jun 2004, Mitsubishi HI) sold to unknown Chinese buyers for USD 51.00 mil, VV Value USD 51.00 mil - BWTS.
