## Main
in Weekly Vessel Valuations Report 31/01/2023
Image reference: 2023-01-31_weekly-vessel-valuations-report-january-31-2023_img1.png
Bulkers : Bulker values have remained stable.
Capesize Aliki (180,200 DWT, Mar 2005, Imabari) sold to unknown Chinese buyers for USD 15.08 mil, VV Value USD 15.77 mil - DD Due.
Supramax Bulk Newport (52,600 DWT, Mar 2003, Kanasashi) sold to undisclosed buyers for USD 9.30 mil, VV Value USD 10.52 mil - SS/DD Due.
Tankers : Tanker values have remained stable
VLCC Birdie (308,800 DWT, Dec 2005, Samsung) sold to Middle Eastern buyers for USD 52.50 mil, VV Value USD 52.98 mil.
VLCC Arcadia V (298,900 DWT, Sep 2000, Kawasaki) sold to undisclosed buyers for USD 40.00 mil, VV Value USD 40.95 mil.
Suezmax Amoroza (159,200 DWT, Dec 2001, Hyundai Heavy Ind) sold to undisclosed buyers for USD 27.00 mil, VV Value USD 25.93 mil - BWTS.
Aframax Kythira Warrior (115,400 DWT, Aug 2006, Sanoyas) sold to undisclosed buyers for USD 38.60 mil, VV Value USD 38.13 mil - BWTS.
LR1s Alpine Pearl, Alpine Pacifica and Alpine Plymouth (74,600 DWT, Jan-Oct 2011, Hyundai Mipo) sold to Torm in an en bloc deal for USD 98.40 mil, VV Value USD 96.89 mil - BWTS.
MR2 Proodos (47,100 DWT, Jan 2005, STX Offshore) sold to Union Maritime for USD 18.70 mil, VV Value USD 17.61 mil - DD Passed.
Containers : Container values have softened
Sub Panamax Cont SITC Surabaya (2,741 TEU, Jun 2006, Aker Ostsee) sold to undisclosed buyers for USD 13.00 mil, VV Value USD 14.15 mil.
