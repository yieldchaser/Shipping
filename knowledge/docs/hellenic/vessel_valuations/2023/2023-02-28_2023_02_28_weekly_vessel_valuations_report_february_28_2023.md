## Main
in Weekly Vessel Valuations Report 28/02/2023
Image reference: 2023-02-28_weekly-vessel-valuations-report-february-28-2023_img1.png
Bulkers : Bulker values have firmed
Capesize Agia Trias (185,800 DWT, Jul 2002, Kawasaki) sold to unknown Chinese buyers for USD 12.50 mil, VV Value USD 12.63 mil - SS Due.
Capesize Navios Felix (181,200 DWT, Sep 2016, Imabari) sold to Navios Maritime Partners LP for USD 40.70 mil, VV Value USD 39.36 mil - Scrubber.
Supramax Pacific Dream and Pacific Vision (56,500 DWT, Jul 2013-Aug 2013, Jiangsu Hantong Ship Heavy Industry) sold to undisclosed buyers in an en bloc deal for USD 31 mil, VV Value USD 29.98 mil - SS/DD Due.
Supramax Kai Xuan 11 (50,200 DWT, Sep 2000, Mitsui Tamano) sold to undisclosed buyers for USD 7.20 mil, VV Value USD 8.44 mil - BWTS.
Handy BC (Open Hatch) Interlink Priority (38,700 DWT, Sep 2015, Taizhou Kouan Shipbuilding) sold to Tufton Oceanic Assets for USD 19.80 mil, VV Value USD 17.57 mil - BWTS.
Tankers : Suezmax values have firmed, Aframaxs have softened
Aframax Delaware Star and Galveston Star (115,000 DWT, Apr 2023, DH Shipbuilding) sold to GNMTC in an en bloc deal for USD 152 mil, VV Value USD 156.74 mil - Resale.
MR2 Marlin Amethyst (50,000 DWT, Jul 2015, CSSC OME) sold to Union Maritime for USD 33.90 mil, VV Value USD 33.90 mil.
MR2 Ibis Pacific (50,000 DWT, Jun 2007, STX Offshore) sold to undisclosed buyers for USD 22.00 mil, VV Value USD 20.84 mil - SS/DD Passed.
Containers : Smaller tonnage Container values have firmed, larger has softened
Sub Panamax Windermere (2,872 TEU, Apr 2010, Yangfan Zhoushan) sold to undisclosed buyers for USD 17.00 mil, VV Value USD 14.63 mil - SS/DD Passed.
Handy Container Julius (1,368 TEU, Nov 2009, Jinling Shipyard Weihai) sold to Lohmann Hermann for USD 9.00 mil, VV Value USD 9.26 mil.
