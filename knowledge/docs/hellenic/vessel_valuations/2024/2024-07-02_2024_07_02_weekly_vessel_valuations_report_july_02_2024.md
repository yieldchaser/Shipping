## Summary
Main
Bulkers: S&P Values Bulker values increased for Capesize and Panamax vessels owing to sales of vessels such as Courageous and Livia Rose
Capesize BC Courageous (181,000 DWT, Nov 2016, SWS) sold to Thenamaris for USD 50.5 mil, VV Value USD 49.65 mil
Panamax BC Livia Rose (81,800 DWT, Feb 2018, Tsuneishi Zhoushan) sold to undisclosed buyers for USD 35.00 mil, VV Value USD 33.78 mil
Supramax BC Tai Hunter (55,400 DWT, Dec 2007, Oshima) sold to unknown Chinese buyers for USD 14.6 mil, VV Value USD 14.27 mil
Tankers: Tanker values remained mostly stable apart from a decrease for older tonnage VLCC, owing to the sale of C Prosperity
VLCC C Prosperity (313,900 DWT, May 2009, Hyundai HI Ulsan Shipyard) sold to Kyklades Maritime for USD 53 mil (SS/DD Due), VV Value USD 57.05 mil
Aframax Pusaka Java (108,500 DWT, Jan 2018, Tsuneishi Zosen) sold to Eastern Pacific Shipping for USD 68.5 mil, VV Value USD 68.25 mil
Containers: Secondhand values continued to spike from firmed earnings, with impressive gains for 15YO Panamax in response to sales
No notable sales
