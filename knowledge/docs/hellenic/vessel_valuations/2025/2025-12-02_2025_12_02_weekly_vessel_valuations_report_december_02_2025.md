## Main
in Weekly Vessel Valuations Report 02/12/2025
Image reference: 2025-12-02_weekly-vessel-valuations-report-december-02-2025_img1.png
Bulkers: Second-hand values largely firmed in the bulker sector, with mid-age Panamax and Handysize bulk carriersshowing substantial gains. Supramax values dipped slightly, following a week of softer sale prices
Panamax Key Frontier (80,700 DWT, 2011, Universal) sold SS/DD Passed for USD 18.2 mil, VV Value USD 16.73 mil
2x Ultramax (66,000 DWT, 2025-2026, Jiangsu Yangzi-Mitsui) sold in an enbloc deal to Precious Shipping for USD 74.5 mil, VV Value USD 75 mil
Handysize Trawind Dolphin (33,700 DWT, 2012, Shin Kurushima) sold for USD 13.8 mil, VV Value USD 13.63 mil
Handysize COSCO Wuyishan (32,000 DWT, 2010, Fujian Mawei) auctioned for USD 8.08 mil, VV Value USD 8.11 mil
Tankers: VV MR Tanker values have firmed this week especially as S&P activity has increased including a number of MR2 en bloc deals
VLCC Mermaid Hope & Mercury Hope (297,200 DWT, 2011, Universal) sold in an en bloc deal by Cido Shipping for USD 120 mil, VV Value USD 124.94 mil
Suezmax Sonangol Namibe (158,400 DWT, Mar 2007, Daewoo) sold to Unknown Greek buyers for USD 34 mil, VV Value USD 34.1 mil
MR2 (Chemical/Product) Hafnia Libra & Hafnia Phoenix (52,400 DWT, 2013, Guangzhou Shipyard International Company Limited) sold in an en bloc deal by Hafnia for USD 46 mil, VV Value USD 47.67 mil
MR2 (Chemical/Product) UOG Syros (51,700 DWT, Aug 2010, Hyundai Mipo) sold SS/DD Passed by United Overseas Group for USD 21.5 mil, VV Value USD 21.01 mil
MR2 (Chemical/Product) PS Queen & PS Dream (51,200 DWT, Aug 2006, STX Offshore) sold in an en bloc deal to Unknown UAE buyers for USD 28 mil, VV Value USD 27.77 mil
Containers: Older Feedermax values are up this week, alongside Handy containers as second-hand demand for smaller vessels remains strong
Sub Panamax Container Happy Lucky (2,553 TEU, Feb 2008, Naikai Setoda) sold DD Passed to Chinese buyers for USD 27.5 mil, VV Value USD 26.41 mil
Handy Container Borkum (1,930 TEU, Oct 2023, Guangzhou Wenchong) sold to Chinese buyers for USD 36.9 mil, VV Value USD 35.9 mil
